• README_Dashboard.md - Guide d'utilisation détaillé

🚀 COMMENT LANCER LE DASHBOARD :

OPTION 1 - Double-clic (le plus simple) :
1. Double-cliquez sur "lancer_dashboard.bat" dans le dossier
2. Le dashboard s'ouvrira automatiquement dans votre navigateur

OPTION 2 - PowerShell :
1. Clic droit dans le dossier → "Ouvrir dans PowerShell"
2. Tapez : .\lancer_dashboard.ps1
3. Appuyez sur Entrée

OPTION 3 - Invite de commande :
1. Ouvrez l'invite de commande dans ce dossier
2. Tapez : "C:/Users/kadem/AppData/Local/Programs/Python/Python313/python.exe" -m streamlit run yshop_dashboard.py
3. Appuyez sur Entrée
